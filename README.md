## 《C++ Primer》第五版中文版习题答案

### 概述
C++ primer第五版 主要用来学习C++语言的细节，书中包含了大量的东西是平时自己不会注意也不会的，学习怎么可以迷迷糊糊呐，希望能完整的从头开始学习，认真的完成每一章习题作业，其实很多在书中已经有答案和对应的example。希望大家也能多学习，多实践。
> 实践是检验真理的唯一标准

* 学习过程中，有反复的认真阅读书籍内容，代码以及习题内容是个人写的；但是有参阅互联网的内容，如果有直接照搬的，会在引用声明
